# Atlas-Hub 実装完了確認 (Update)

## 変更内容
- **進捗表示機能 (`ProgressView`) の追加**:
    - 商品ごとに「本体」と「鞘」のNC粗削/仕上時間を表示するバーを追加しました。
    - 実績時間（ログ）と標準時間（マスタ）比較し、超過時は警告を表示します。
- **ロジック更新 (`inventory.py`)**:
    - ログから `Process` (Rough/Finish) と `Time` を抽出し、平均時間を算出するようにしました。
- **データ前提**:
    - **マスタ**: `本体NC粗削時間`, `本体NC仕上時間`, `鞘NC粗削時間`, `鞘NC仕上時間`
    - **ログ**: `Process` (Rough/Finish), `Time`

## 検証方法
1. `atlas-hub` フォルダ内で `streamlit run app.py` を実行します。
2. データファイルに該当するカラムが含まれている場合、詳細パネルに進捗バーが表示されます。
